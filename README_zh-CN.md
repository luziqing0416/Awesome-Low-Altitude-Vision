# 低空视觉资源精选

> 面向低空无人机视觉的数据集、论文、基准与研究资源索引。

[English](README.md) | **简体中文**

本仓库以**应用场景作为一级分类**，以视觉任务和传感器模态作为标准化标签，并在同一场景内按**发布年份倒序**排列。这样既能突出搜救与灾害响应，又能保留巡检、交通、农业和生态等可能为研究提供启发的数据。

## 排序与链接标准

1. 类别按应用场景划分，数据集只放在最主要的场景中，其他用途通过标签表达。
2. 每一类内部按发布年份从新到旧排序。
3. 链接优先级为：官方/作者维护页面 → 出版社或 CVF/arXiv 论文页 → Zenodo 等可信存档 → 明确标注的社区镜像。
4. 对真实、合成和混合数据进行明确标记，同时记录下载是否开放、是否需要申请。
5. 对搜救数据重点记录连续视频、无人目标负帧比例、目标尺度、采集是否真实或人为构造等信息。

## 当前重点

首批条目已经覆盖 SeaDronesSee、NOMAD、HERIDAL、RescueNet、FloodNet、FLAME、TTPLA、UAVDT、VisDrone、BIRDSAI、CART 等数据集。

UAVDT 原先无法访问的 Google Sites 入口已更换为稳定的 [Zenodo 数据存档](https://zenodo.org/records/14575517) 和 [ECCV 论文页](https://openaccess.thecvf.com/content_ECCV_2018/html/Dawei_Du_The_Unmanned_Aerial_ECCV_2018_paper.html)。

## 收录原则

巡检、农业、交通和生态数据虽然不是当前搜救研究的核心，但会保留，因为其中的小目标、复杂背景、跨模态、域迁移、长尾分布和精细标注问题可能具有借鉴价值。纯卫星遥感数据原则上不收录。

本仓库只提供索引，不重新分发数据集和受版权保护的论文 PDF。各资源仍遵循原始许可证和使用条件。
